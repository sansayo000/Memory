# Memory
# 3歳向け 神経衰弱スマホゲーム 仕様書
（仮称：カードあそび）

---

# 1. 目的

## 背景
外出先（待ち時間・移動中）に、3歳児が短時間・安全に遊べる知育ゲームが欲しい。

## 目的
- 記憶力・集中力・語彙の発達を促す
- 親子で一緒に遊べる
- スマホ1台で完結
- オフラインでも動作可能
- ブラウザのみで配布（GitHub Pages）

---

# 2. ターゲット

## メイン
- 3歳児（2.5〜4歳）

## サブ
- 保護者
- 4〜6歳の兄姉

## ユーザー特性（重要）
- 文字はほぼ読めない
- タップ精度が低い
- 負け続けると癇癪になりやすい
- 集中時間は1〜3分程度

### 設計方針
👉 ストレスゼロ・簡単すぎるくらいが最適

# 5. 画面仕様

---

## 5.1 トップ画面

### UI要素
- タイトル
- 協力プレイボタン（大）
- 対戦プレイボタン（大）
- 設定ボタン（⚙）
- 音量ボタン（🔊）

### UIルール
- ボタン最小80px以上
- アイコン中心（文字最小）
- 1タップ開始

---

## 5.2 ゲーム画面

### レイアウト
| 難易度 | グリッド |
|-----------|------------|
| 6枚 | 3×2 |
| 8枚 | 4×2 |
| 12枚 | 4×3 |

### 表示要素
- カード一覧
- 手数表示（小さめ）
- 戻るボタン
- ヒントボタン（任意）

### カード状態
| 状態 | 挙動 |
|--------|----------------|
| 裏 | タップ可能 |
| 表 | タップ不可 |
| 一致 | 固定 or 消える |

### 判定フロー
1. 1枚目めくる
2. 2枚目めくる
3. 一致 → 固定
4. 不一致 → 1秒後に裏返す
5. 全一致 → クリア

---

## 5.3 クリア画面

### 表示内容
- 「やったね！」テキスト
- 星評価（★1〜3）
- もう一回ボタン
- ホームへ戻る

### 演出
- 紙吹雪アニメーション
- 成功音再生

---

## 5.4 設定画面

### 項目
- カード枚数
- 音ON/OFF
- モード初期値
- データリセット

保存：localStorage

# 6. モード仕様

## 協力モード（デフォルト）
- 勝敗なし
- 全部揃えたらクリア
- 手数のみ表示
- 失敗ペナルティなし

👉 3歳向け基本モード

---

## 対戦モード
- 交互ターン制
- 取った枚数で勝敗決定
- プレイヤー表示あり

👉 親子・兄弟向け

---

# 7. UX設計ルール（重要）

## やってはいけない
- 小さいボタン
- 長い文章
- 制限時間
- 厳しい負け演出
- 複雑操作

## 必須
- 大きなタップ領域
- 派手な成功演出
- すぐ再挑戦可能
- 短時間クリア（3分以内）
- 読めなくても理解可能
